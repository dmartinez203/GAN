üìã Project Overview

Problem Statement

Fake images have become ubiquitous on social media, affecting public discourse. This project demonstrates how Generative Adversarial Networks (GANs) can create synthetic images that look like real ones, helping understand the technology behind fake image generation.

üéØ Key Features

Generator Network

Discriminator Network

Binary classifier to distinguish real from fake images

MNIST Dataset

60,000 handwritten digit images for training

Adversarial Training

Alternating optimization of Generator and Discriminator

üî¨ Algorithm Overview

Generator Network Architecture

  • Input: 100-dimensional noise vector (Gaussian distribution)
  • Layer 1: Dense(256) ‚Üí LeakyReLU(0.2) ‚Üí BatchNormalization
  • Layer 2: Dense(512) ‚Üí LeakyReLU(0.2) ‚Üí BatchNormalization
  • Layer 3: Dense(1024) ‚Üí LeakyReLU(0.2) ‚Üí BatchNormalization
  • Output: Dense(784) ‚Üí Tanh ‚Üí Reshape(28√ó28√ó1)

Discriminator Network Architecture

  • Input: 28√ó28 grayscale image
  • Layer 1: Dense(1024) ‚Üí LeakyReLU(0.2) ‚Üí BatchNormalization
  • Layer 2: Dense(512) ‚Üí LeakyReLU(0.2) ‚Üí BatchNormalization
  • Layer 3: Dense(256) ‚Üí LeakyReLU(0.2) ‚Üí BatchNormalization
  • Output: Dense(1) ‚Üí Sigmoid (real/fake probability)

üéì Training Process

  1. Load Data: MNIST dataset (60,000 images), normalized to [-1, 1]
  2. Train Discriminator: On real images (label=1) and fake images (label=0)
  3. Train Generator: Through GAN model to maximize Discriminator error
  4. Repeat: Alternate training for 400+ epochs
  5. Evaluate: Visual inspection at milestones (epochs 1, 30, 100, 200, 400)

üìä Performance Analysis

After 400 epochs of training:

  • Early epochs (1-30): Noisy, unrecognizable outputs
  • Mid epochs (30-100): Digit shapes emerge, blurry details
  • Late epochs (200-400): Clear, recognizable digits with minor artifacts
